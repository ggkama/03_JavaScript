함수명([전달인자...]); // 함수 호출

전달 인자(Argument), 매개 변수(Parameter)

* 전달 인자(Argument)
  - 함수"호출"시 ()안에 작성되는값
  - 작성된 값은 함수의 매개 변수로 전달됨

* 매개 변수(Parameter)
  - 함수"선언"시 ()안에 작성되는 변수
  - "호출"시 전달되는 값(전달 인자)를 저장하기 위한 변수
  - 매개 변수는 함수의 지역 변수(함수에서만 사용 가능)

return(반환)

- 함수 내에 작성하는 명령어

- 현재 수행중인 함수를 종료하고
  호출한 곳으로 돌아가라

- return;  -> 함수종료후, 호출한곳으로 리턴

- return 값; -> 함수종료후, 값을 가지고 호출한곳으로 리턴

(참고)
함수 정의 시 return을 작성하지 않은 경우
자동으로 함수 정의 끝 부분에 return 구문이 추가된다.

화살표 함수(Arrow function)

- 익명 함수를 -> (화실표) 기호를 이용해서 간단히 작성하는 방법

  익명함수 : function(){}

  1. 기본형태 : () +> {}

  2. 매개 변수가 1개인 경우 : 매개변수가 1개일 경우 소괄호 생략 가능
      (e) => {}
      또는
      e => {}
  
  3. 매개 변수가 여러개인 경우 : 소괄호 생략 x
      (a,b,c) => {}

  4. 함수 정의 부분{}에 한줄만 있을경우 {} 생략가능 

    () => { console.log(123)}
    또는
    () => console.log(123) 

    * {} 내 작성된 한 줄이 return 구문인 경우  return 생략 가능

      ()=> {return 1+2;} 인 경우
      또는
      ()=> 1+2;

  5. 함수 정의 부분에 return 한줄만 있는데,
      return되는 값이 object타입인 경우(배열, 객체)  
      {},return 생략 불가능

      () => { return [1,2,3];}

      () => { return {name : "홍길동"}}
